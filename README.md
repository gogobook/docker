# PostgreSQL 大補帖
基於 [PostgreSQL Official Docker Repository](https://hub.docker.com/_/postgres/)，增加安裝各式常用延伸功能，方便入門者進行功能試用。

## Docker Installation
本專案需要您準備好 Docker 環境，並瞭解基本操作方式。\
Ubuntu 的 Docker 安裝方式，請參閱 Docker-install 目錄內容。
